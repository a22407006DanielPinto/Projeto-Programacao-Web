O que é o Django MVT?

Este projeto segue o padrão de arquitetura Model-View-Template:

  • Model: Gere os dados e a base de dados (ex: Projetos e Tecnologias).
  • View: Contém a lógica que decide o que mostrar ao utilizador.
  • Template: É a parte visual em HTML que o utilizador vê.
